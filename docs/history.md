# v1 → v2 転換の経緯

v1(MVP)は「通称⇔公称の地図ラベル切り替え」を目的として作られたが、実際に解決したい課題は
「旧住所や通称しか分からず、郵便物が現在の正式住所に届かない」という事象であることが
明確になった。大分市では住居表示の実施により住所が順次変更されており(令和7年11月時点で
38地区)、通称・地図ラベルの表示だけでは根本課題は解決しない。

そこで v2 では、コア機能を「地図ラベルの切り替え」から「旧住所・通称・新住所いずれからでも
現在の正式住所を検索できる検索ツール」へ転換した。地図は検索結果の該当地区を示す補助的な
役割に位置づけを変えている。v1 のトグル UI は撤去したが、`src/data/chomei.geojson` は
削除せずリポジトリに残している。

詳細は [spec-mvp.md](spec-mvp.md)(v1)と [spec-v2-address-lookup.md](spec-v2-address-lookup.md)(v2)を参照。
